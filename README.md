## Installation requirements.
+ ReactJS v15.3.0
+ Bootstrap 3
+ create-react-app tool
+ react-router 2.6.1 


## Instructions:

+ Swap out the firebase url in app.js with your own
+ npm install
+ npm start
+ Visit localhost:3000
